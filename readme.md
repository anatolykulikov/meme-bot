# МемБот
В жизни каждой команды может наступить момент, когда она не сможет войти в нужную дверь. Или сервис. Или просто созреет до того, то рабочие вопросы решаются в закрытой специальной экосистеме, а не обычном мессенджере.

Но чтобы жизнь была не скучной и обычной, можно использовать мем-бота, который даст предсказание на сегодняшний день.

Данный репозиторий — базовая база для такого бота. Здесь нет никаких баз данных, хранилища и прочего — только самое нужное. Остальное можно добавить по вкусу =)

## Тестирование

### Mattermost
Для локального тестирования мема предсказания использовать данный запрос:
```bash
curl -X POST http://localhost:3000/mem \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "user_id=test_user_123" \
  -d "user_name=alex" \
  -d "channel_id=channel123" \
  -d "team_id=team123" \
  -d "text="
```
